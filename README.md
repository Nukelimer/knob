## Description :üí°:

Knob is a full-stack e-commerce solution that leverages SvelteKit for both server-side API logic and a highly interactive frontend. Designed to handle dynamic product galleries, promotional banners, and fast checkouts, it uses GSAP and Embla Carousel for premium product displays, all styled with the lightning-fast Tailwind CSS v4 engine.

### Project Scripts:

- `pnpm dev`: Start development server and backend API routes
- `pnpm build`: Build the full-stack application for production
- `pnpm check`: Run Svelte-check and TypeScript validation
- `pnpm lint`: Run ESLint and Prettier formatting
